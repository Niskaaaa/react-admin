# 项目简介

react-admin是一个前后台分离的后台管理SPA项目

包括用户管理、商品分类管理、商品管理、权限管理等功能

前端使用react全家桶+antd4.0+axios+ES6+webpack等技术

后端方面采用python+flask+mongodb
